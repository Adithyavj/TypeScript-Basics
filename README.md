# TypeScript Basics
- To install typescript:
```
    npm install typescript
```

- To compile a typescript file to a javascript file we use,
```
    tsc <typescript-filename>
```
- To compile all typescript files to js,
```
    tsc
```
- Compile with watch
```
    tsc --watch
```
- To create a typescript config file
```
    tsc --init
```
- To run a js file
```
    node <filename>
```
